## Featured Project

### Wayfinder

An OSINT-driven asset ownership and infrastructure mapping tool.

Built to answer:

**“Who actually owns this?”**

Wayfinder uses DNS intelligence, ASN ownership, MX analysis, nameserver patterns, and weighted confidence scoring to produce explainable ownership assessments instead of raw data dumps.

It helps reduce ambiguity during:

- Attack surface mapping
- Bug bounty reconnaissance
- Vendor / third-party validation
- Corporate asset ownership review
